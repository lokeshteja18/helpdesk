## ğŸ“Š Data Query Examples (SQL-like for visualization)

```
1. USER DASHBOARD QUERIES:

   GET Total Tickets for User:
   SELECT COUNT(*) FROM tickets WHERE userId = 4
   Result: 12

   GET Open Tickets:
   SELECT COUNT(*) FROM tickets WHERE userId = 4 AND status = 'open'
   Result: 5

   GET Recent Tickets:
   SELECT * FROM tickets WHERE userId = 4 ORDER BY createdAt DESC LIMIT 5


2. AGENT DASHBOARD QUERIES:

   GET Assigned Tickets:
   SELECT * FROM tickets WHERE assignedTo = 3

   GET Resolved Count:
   SELECT COUNT(*) FROM tickets WHERE assignedTo = 3 AND status = 'closed'
   Result: 3

   GET Performance Metrics:
   SELECT COUNT(*) as total,
          SUM(CASE WHEN status='closed' THEN 1 ELSE 0 END) as resolved,
          SUM(CASE WHEN status='in-progress' THEN 1 ELSE 0 END) as pending
   FROM tickets WHERE assignedTo = 3


3. ADMIN DASHBOARD QUERIES:

   GET All Tickets with Assignee:
   SELECT t.*, u.name as agentName
   FROM tickets t
   LEFT JOIN users u ON t.assignedTo = u.id
   ORDER BY t.createdAt DESC

   GET Agent Performance:
   SELECT u.id, u.name, COUNT(t.id) as assigned,
          SUM(CASE WHEN t.status='closed' THEN 1 ELSE 0 END) as resolved
   FROM users u
   LEFT JOIN tickets t ON u.id = t.assignedTo
   WHERE u.role = 'agent'
   GROUP BY u.id


4. SUPER ADMIN DASHBOARD QUERIES:

   GET User Breakdown:
   SELECT role, COUNT(*) FROM users GROUP BY role

   GET System Statistics:
   SELECT (SELECT COUNT(*) FROM users) as totalUsers,
          (SELECT COUNT(*) FROM users WHERE role != 'user') as staffMembers,
          (SELECT COUNT(*) FROM tickets) as totalTickets,
          (SELECT COUNT(*) FROM tickets WHERE status = 'open') as openTickets
```
